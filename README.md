      <h3>Urumuri Pharma — Live Pharmacy Inventory & Orders Dashboard</h3>
      <p>
        Urumuri Pharma is a live pharmacy operations dashboard built to manage inventory,
        capture customer orders, update stock levels immediately, and record each order as a
        sales transaction in the backend. The system includes a FastAPI backend, SQLite
        database, and a simple frontend dashboard designed for pharmacists and pharmacy staff.
      </p>
      <p>
        The dashboard displays live business KPIs, current inventory levels, reorder status,
        product catalogue, recent orders, and an order entry form. It also enforces an important
        business rule that prevents the total sales for any calendar month from exceeding
        <b>RWF 15,000,000</b>.
      </p>
      <p>
        <b>Technologies Used:</b><br/>
        Python, FastAPI, SQLite, HTML, CSS, JavaScript
      </p>
